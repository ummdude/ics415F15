Initial simple idea is to create a web page for a user to read and write basic simple notes quickly. A user may create multiple notes with perhaps some limit.

The base initial features include:

  • special markers that a user could jump to while editing/viewing their text
  • double columns for optimal usage of window space
  • resizeable dual view of the same or 2 doumentsin one window

Possible features include:

  • tabbed files, like excel has multiple sheets
  • simple spreadsheets with basic functions implemented
  • html, js, cs writing and viewing

    • Site concepts taken from:

  • Google Docs, paste bin, wikipedia, spreadsheet software, text document software
